`nflstats` is a tool developed to test consuming [sportsdata.io](https://sportsdata.io) NFL feeds.

## Installation

```sh
go install github.com/tomshaw/nflstats@latest
```

> nflstats has been tested and supports Go versions >=1.24.x.

## API Key

First of all, obtain your Free Trial API Key from the [sportsdata.io](https://sportsdata.io) developer portal.

## Configuration

In order to use `nflstats`, create a `.env` file in the root of the project and add your API key.

```sh
API_KEY=API_KEY_GOES_HERE
```

## Usage

Run `nflstats -h` to print help instructions.

---

## Api Endpoints

Run `nflstats list` to print available API's. Please submit a pull request if you would like 
to add addtional endpoints.

```

``` 

## News By Team

Simple specify the official team abbrevation `dal`, `ari`, `buf`, `phi`.

> All API requests support Json, Pretty Print or Table export options.

```sh
nflstats -table -api NewsByTeam -team dal
```
